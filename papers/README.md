# Downloaded Papers

This folder contains papers gathered for:
**Probing the Limits: Adversarial and OOD Robustness of Neural Thicket Ensembles**.

## Primary Papers (downloaded)

1. **Neural Thickets: Diverse Task Experts Are Dense Around Pretrained Weights** (`2603.12228_Neural_Thickets_Diverse_Task_Experts_Are_Dense_Around_Pretrained_Weights_2026.pdf`)
   - Authors: Yulu Gan, Phillip Isola
   - Year: 2026
   - arXiv: 2603.12228
   - Why relevant: Core method family for thicket-style random perturbation experts.

2. **Robust Deep Learning Ensemble against Deception** (`2009.06589_Robust_Deep_Learning_Ensemble_against_Deception_2020.pdf`)
   - Authors: Wenqi Wei, Ling Liu
   - Year: 2020
   - arXiv: 2009.06589
   - Why relevant: Joint adversarial + OOD defense via diversity-oriented ensemble verification.

3. **Ensemble Adversarial Training: Attacks and Defenses** (`1705.07204_Ensemble_Adversarial_Training_Attacks_and_Defenses_2017.pdf`)

   - Year: 2017
   - arXiv: 1705.07204
   - Why relevant: Foundational ensemble-based adversarial training baseline.

4. **Simple and Scalable Predictive Uncertainty Estimation using Deep Ensembles** (`1612.01474_Simple_and_Scalable_Predictive_Uncertainty_Deep_Ensembles_2016.pdf`)

   - arXiv: 1612.01474

5. **Out-of-Distribution Detection Using an Ensemble of Self Supervised Leave-out Classifiers** (`1809.03576_OOD_Detection_Ensemble_Self_Supervised_Leave_out_Classifiers_2018.pdf`)

6. **Enhancing Certifiable Robustness via a Deep Model Ensemble** (`1910.14655_Enhancing_Certifiable_Robustness_via_Deep_Model_Ensemble_2019.pdf`)

   - Year: 2019

7. **Certifying Joint Adversarial Robustness for Model Ensembles** (`2004.10250_Certifying_Joint_Adversarial_Robustness_for_Model_Ensembles_2020.pdf`)
   - Authors: Mainuddin Ahmad, David Evans
   - Year: 2020
   - arXiv: 2004.10250
   - Why relevant: Formal certification for joint robustness of ensembles.

8. **PEP: Parameter Ensembling by Perturbation** (`2010.12721_PEP_Parameter_Ensembling_by_Perturbation_2020.pdf`)
